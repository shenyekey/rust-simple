# Rust 编程之旅

《Rust 编程之旅》已经完结啦，感谢大家的关注和支持！合集视频内容，旨在用简单思路，解释 Rust 的基础语法知识。

> 声明: Rust 编程之旅合集视频均为博主自己录制，非搬运。

## 1. 基础内容

- 01 ~ 09 ，rust 开发工具、常量和函数；
- 10 ~ 19 ，rust 控制语句和引用的用法；
- 20 ~ 29 ，rust 枚举、结构和方法的用法；
- 30 ~ 39 ，rust 工程化，目录结构、包、模块和测试的用法；
